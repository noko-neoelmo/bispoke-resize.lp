# Claude Code 指示書：RETOLD TOKYO 指輪サイズ直しLP ヒーロー改善

対象リポジトリ：`noko-neoelmo/bispoke-resize.lp`

## 目的

RETOLD TOKYOの指輪サイズ直しLPのファーストビューを、スマホ最適で改善してください。

優先目的は以下です。

- スマホ閲覧時のCVR改善
- 「LINEで無料相談する」CTAのクリック率向上
- 「0.5号刻み」の強みをファーストビューで明確に伝える
- 「他店で断られた指輪も相談できる」ことを強く訴求する
- ホワイトベース＋ネイビーで、現状よりパキッと上品に見せる
- 買取店・チラシ・安売り感を避ける

## 参考画像

画像は以下のパスに配置してください。

```txt
public/images/resize-lp/current-hero-reference.webp
public/images/resize-lp/target-mobile-hero-reference.webp
```

配置前に `docs/lp-redesign/asset-list.md` を確認してください。

## 実装方針

1. スマホファーストで設計する。
2. ファーストビュー内にCTAボタンを必ず表示する。
3. CTAは画面中央〜下部の視線が止まる位置に大きく配置する。
4. CTA文言は「LINEで無料相談する」。
5. CTA下に小さく「ご相談・お見積もりだけでもお気軽にどうぞ」を入れる。
6. メインコピーは「0.5号刻みの 指輪サイズ直し」。
7. サブコピーは「大きくも小さくも、0.5号刻みで丁寧に調整。ブランドリング・石付きリングも対応します。」。
8. ヒーロー内の訴求要素は以下を優先する。
   - 5,500円〜（税込）
   - 全国・往復送料無料
   - 国家資格保有の職人が対応
   - 大きくも小さくも対応OK
9. リングゲージ訴求をCTA直下にカードで配置する。
   - 見出し：「0.5号刻みのリングゲージ 無料送付」
   - 補足：「サイズが分からなくてもOK」
   - リングゲージ画像を使用する。
10. 「他店で断られた指輪もご相談ください」は、ヒーロー直下に大きめの見出しとして配置する。
11. その下に3つの小カードを置く。
    - ブランドリング対応
    - 石付きリング対応
    - 変形・ゆがみがある指輪もOK
12. 配色は白背景、ネイビー文字、CTAのみLINEグリーン。
13. ネイビーは `#071D49` または近い深い紺。
14. CTAグリーンは `#06C755` をベースに、必要なら少し落ち着かせる。
15. 金色はアクセントに限定し、多用しない。
16. 余白を広めに取り、野暮ったい装飾・過剰なバッジ・チラシ風の囲みは避ける。
17. PC表示は崩れない程度でよいが、最優先はスマホ表示。

## 画面構成案

### 1. Header

- ロゴ：RETOLD TOKYO
- 右上に小さめのCTA「LINEで無料相談する」
- ただし主CTAはヒーロー内に置く。ヘッダーCTAは補助扱い。

### 2. Campaign Bar

```txt
まとめ割｜2つ以上のサイズ直しで10%OFF
```

- ネイビー帯
- 文字は白、`10%OFF` のみ控えめなゴールド
- 高さは詰めすぎず、でもスマホで邪魔にならない程度

### 3. Hero

メインコピー：

```txt
0.5号刻みの
指輪サイズ直し
```

サブコピー：

```txt
大きくも小さくも、0.5号刻みで丁寧に調整。
ブランドリング・石付きリングも対応します。
```

ヒーロー画像：

- 手元＋指輪の写真
- 白背景に溶けるように配置
- 工場感、買取店感、過剰な広告感は避ける

ベネフィットアイコン：

```txt
5,500円〜（税込）
全国・往復送料無料
国家資格保有の職人が対応
大きくも小さくも対応OK
```

### 4. Main CTA

- ヒーロー内、アイコン訴求の直下に大きく配置
- スマホ幅では横幅80〜90%程度
- ボタン文言：

```txt
LINEで無料相談する
```

- ボタン下マイクロコピー：

```txt
ご相談・お見積もりだけでもお気軽にどうぞ
```

### 5. Ring Gauge Card

CTA直下に配置。

```txt
0.5号刻みのリングゲージ
無料送付
サイズが分からなくてもOK
```

- 0.5号刻みであることを必ず入れる
- リングゲージ画像を使用
- カードは白背景、薄いゴールドまたは薄いネイビー枠

### 6. Rejected Ring Section

ヒーロー直下、スクロール直後に強く見える位置。

見出し：

```txt
他店で断られた指輪も
ご相談ください
```

小カード：

```txt
ブランドリング対応
石付きリング対応
変形・ゆがみがある指輪もOK
```

## 完成後の確認項目

- iPhone幅 375px / 390px / 430px でファーストビュー内にCTAが見えているか
- 「0.5号刻み」が一瞬で読めるか
- 「他店で断られた指輪も相談できる」がスクロール直後に目立つか
- CTAが1画面目で押したくなる位置にあるか
- 現状より買取店っぽさ、チラシ感が減っているか
- リングゲージが「0.5号刻み」であることが伝わるか

## 注意

LP全体の思想やブランドストーリーを増やしすぎないでください。今回の主目的は、広告流入後のスマホFV改善とCVR改善です。
