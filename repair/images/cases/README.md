# 修理事例 Before/After 画像 — 生成ブリーフ（CodeX用）

`/repair` LP「修理の事例と料金」セクションで使用する Before/After 画像。
現在は既存写真を流用したプレースホルダー。**下記の正しいパスに本番画像を上書き**すれば、HTMLの変更は不要。

## 共通仕様
- アスペクト比: **1:1**（正方形）。推奨 800×800px 以上、JPG。
- 2枚1組（before / after）。同一アングル・同一トリミングで「変化」が伝わるように。
- ブランドフォトディレクション（`bispoke-brand-guidelines` 準拠）:
  - ハードライト＋影を活かす。背景は **ダークネイビー / マットブラック / コンクリート**。
  - マクロ寄り、被写界浅め。彩度は抑えめ（cool neutral）。
  - キラキラ・暖色演出・ソフトフォーカス・グロスはNG。
  - 手を写す場合は日本人女性、ネイルはグレー/ニュートラル。
- before=破損/劣化が分かる状態、after=修理完了で清潔・整った状態。
